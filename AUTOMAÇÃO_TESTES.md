# Automação de Testes (Bônus do desafio)

Este documento descreve a automação implementada como parte opcional do desafio.

## Stack utilizada
- Jest
- Supertest
- NestJS (aplicação executada in-memory)

## Como executar
A automação não exige que o backend esteja rodando separadamente.

```bash
cd backend
npm install
npm run test:e2e
